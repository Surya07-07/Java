# Student Expense Tracker

A simple Java console-based application to record student daily expenses.

## Features
- Add expense (description, category, amount)
- View all expenses
- Generate summary report (total + category-wise)

## Technologies Used
- Java
- OOP (Classes & Objects)

## How to Run
1. Install Java or use BlueJ.
2. Open the project folder.
3. Run `Main.java`.
